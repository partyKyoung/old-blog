C

자료구조나 알고리즘 공부 시 c로 코드를 작성 중.

  1. gcc 설치
brew install gcc
  1. c로 작성 중인 프로젝트 폴더에 tasks.json 파일 작성.
{
  "version": "2.0.0",
  "tasks": [
    {
      "type": "shell",
      "label": "gcc build active file",
      "command": "/usr/bin/gcc",
      "args": [
        "-g",
        "${file}",
        "-o",
        "${fileDirname}/${fileBasenameNoExtension}"
      ],
      "options": {
        "cwd": "/usr/bin"
      },
      "problemMatcher": ["$gcc"],
      "group": {
        "kind": "build",
        "isDefault": true
      }
    }
  ]
}
